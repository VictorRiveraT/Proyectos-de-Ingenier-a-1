# Bocetos del prototipo SafeFry  
## Evaluación de aceite vegetal mediante turbidez óptica y capacitancia dieléctrica  

Durante la etapa de diseño conceptual se desarrollaron tres propuestas de bocetos que integran los dominios eléctrico, mecánico y de software.  
Cada propuesta buscó balancear portabilidad, resistencia térmica y compatibilidad con los sensores seleccionados.  
Finalmente, se eligió el Boceto 1 (ESP32) como versión definitiva para el modelado 3D y construcción del prototipo funcional.

---

## Boceto 1 – Versión final seleccionada (ESP32)  

<p align="center">
  <img src="../Im%C3%A1genes/Boceto%201.jpg" width="600">
</p>

Este diseño representa la propuesta definitiva del dispositivo.  
Su estructura cilíndrica compacta permite alojar los sensores de turbidez y capacitancia dentro de un cuerpo principal fabricado recubierto internamente con enacrílico líquido o resina transparente para protección adicional del prototipo a altas temperaturas (no es recomendable, pero sirve como medida preventiva), materiales resistentes al calor moderado del aceite tibio (30–50 °C).

**Componentes principales**  
- **Microcontrolador:** ESP32 DevKit V1.  
- **Pantalla LCD frontal** para mostrar el estado del aceite (“Apto / No apto”).  
- **Sensor de turbidez sin contacto (SEN0554)**, montado lateralmente con una ventana transparente.  
- **Sensor capacitivo encapsulado en resina (SEN0308)**, ubicado en el eje central, aislado del aceite.  
- **Fuente de energía:** batería ion-litio plana recargable.  
- **Conversor:** módulo Boost Converter para estabilizar voltaje.  
- **Botones pulsadores físicos:** encendido y medición.  
- **Comunicación inalámbrica:** Bluetooth o Wi-Fi para envío de datos al móvil.  

**Materiales y fabricación**  
- Cuerpo principal de acrílico líquido transparente o resina epóxica.  
- Soportes internos impresos en 3D con filamento PETG o ABS (resistentes hasta 85 °C).  
- Sección inferior desmontable para reemplazo de baterías.  

**Ventajas**  
- Portátil y ligero.  
- Compatible con los sensores de la matriz morfológica.  
- Buena resistencia térmica y aislamiento eléctrico.  
- Estructura ópticamente adecuada para la lectura del sensor de turbidez.  
- Fabricable con los recursos del laboratorio de ingeniería UPCH.

## Boceto 2 – Versión con Raspberry Pi 5  

<p align="center">
  <img src="../Im%C3%A1genes/Boceto%202.jpg" width="600">
</p>

Este diseño explora un enfoque más avanzado, orientado a procesamiento local de datos y almacenamiento interno.  
Mantiene una forma cilíndrica pero incorpora mayor complejidad de hardware.

**Componentes principales**  
- Raspberry Pi 5 como unidad de procesamiento.  
- Pantalla OLED de alta definición.  
- Sensor de turbidez sin contacto doble.  
- Sensor capacitivo vertical encapsulado.  
- Mini batería de níquel-litio recargable.  
- Módulo Buck Converter y sistema de encendido con retención.  

**Observaciones**  
- Permite procesamiento de datos en el dispositivo.  
- Aumenta el costo, consumo energético y dificultad de ensamblaje.  
- Recomendado para versiones de laboratorio, no para un dispositivo portátil.

---

## Boceto 3 – Versión con Arduino UNO  

<p align="center">
  <img src="../Im%C3%A1genes/Boceto%203.jpg" width="600">
</p>

Prototipo inicial orientado a pruebas de calibración y validación de sensores.  
Tiene estructura rectangular vertical, con compartimentos separados para cada módulo.

**Componentes principales**  
- Arduino UNO como controlador principal.  
- Pantalla TFT táctil para visualización e histórico.  
- Botón capacitivo (touch) de encendido.  
- Sensor de turbidez analógico modificado.  
- Sensor capacitivo encapsulado en resina.  
- Batería ácido-plomo recargable.  
- Módulo Buck-Boost para regulación de energía.  
- App móvil para iniciar medición mediante Bluetooth/Wi-Fi.  

**Observaciones**  
- Diseño robusto pero voluminoso.  
- Ideal para prototipos de laboratorio o pruebas de validación.  
- No óptimo para portabilidad.

---

## Conclusión  

Tras la evaluación de los tres bocetos, se seleccionó el Boceto 1 (ESP32) como diseño final.  
Este modelo cumple con los criterios establecidos en la matriz morfológica: portabilidad, eficiencia energética, resistencia térmica moderada y compatibilidad con los sensores óptico y capacitivo seleccionados.  
Además, su estructura cilíndrica y modular es adecuada para la fabricación mediante impresión 3D y el uso de acrílico líquido disponible en el laboratorio de ingeniería UPCH.  
Este será el modelo base para el diseño 3D y la fabricación del prototipo físico de SafeFry.
