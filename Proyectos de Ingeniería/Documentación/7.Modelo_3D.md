# Modelo 3D del Prototipo SafeFry

Link de OneShape: https://cad.onshape.com/documents/33e2e4672be8dc7ec5e8ae09/w/13b6d7be983be934564895bf/e/6dd204cf2977ec522326948d

- El modelo tridimensional del prototipo SafeFry fue desarrollado en Onshape para representar el diseño físico del dispositivo medidor de calidad de aceite vegetal.  
- El conjunto está conformado por tres piezas principales: la carcasa, el porta-muestra, tapa superior y lateral.  
- Cada una se diseñó para cumplir una función específica y facilitar el ensamblaje del circuito y los sensores.

---

## 1. Carcasa principal

La carcasa contiene la base donde se fija el sistema electrónico. En su interior se incluyen soportes para el PCB, la batería y los sensores, además de orificios de sujeción y espacio para el cableado.  
El diseño busca ser compacto, modular y fácil de imprimir en 3D.

Mantiene alineadas las sondas del sensor de turbidez y el sensor capacitivo, asegurando una medición estable y repetible.

## 3. Tapa superior

La tapa protege los componentes internos y permite cerrar el sistema de forma segura.  
Incluye un acople lateral que cubre el espacio del porta-muestra y los sensores, manteniendo el dispositivo sellado pero accesible para mantenimiento o calibración.

## 5. Comentarios finales

El modelado 3D permitió validar la compatibilidad entre las piezas, verificar los espacios internos y optimizar el ensamblaje antes de la impresión.  
Con este diseño se garantizó la correcta integración del sistema electrónico, los sensores y la estructura del prototipo SafeFry.
